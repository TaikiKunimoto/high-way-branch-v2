ユーザーへの問いかけ（AskUserQuestion の設問・選択肢含む）や連絡・進捗報告は**日本語**で伝える。

**Why:** ユーザーが「私に問いかける時などは日本語で伝えてね」と明示。
**How to apply:** 質問・確認・ステータス報告など user-facing なテキストは日本語で書く。コード・コミットメッセージ・PR本文は従来どおり（このリポジトリは日本語コメント/コミットが既定 → [[class-centric-organization]] 周辺の既存スタイルに合わせる）。
